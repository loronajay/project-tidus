# Project Tidus

## Subsystem Procurement Sheet

## 1. Purpose

This document defines what is needed to start Project Tidus in a way that produces real progress without creating unnecessary instability, wasted spend, or early scope drift.

It is intentionally structured around the build plan:

- First establish a stable electric baseline
- Then establish a stable bench integration environment
- Then develop the cartridge and hydrogen packaging layer
- Only then move into controlled hydrogen commissioning

## 2. Procurement Principles

Procurement for Tidus should follow these rules:

- Buy only what supports the next validation gate
- Prefer off-the-shelf, well-understood hardware in the first build
- Do not buy advanced hydrogen hardware before the battery-only and bench-test phases are ready to use it
- Do not fabricate around unknown dimensions that can be mocked first
- Favor parts with clear ratings, documentation, and replacement availability
- Stability is more valuable than novelty

## 3. What You Need To Get Started

To begin making real progress, you need five things:

1. A bicycle platform suitable for a rear-hub e-bike conversion
2. A basic 48V electric propulsion baseline
3. A bench-test workspace for electrical integration
4. Safety and measurement tools
5. A documentation workflow for recording tests and decisions

If those exist, the project can begin producing meaningful results even before hydrogen hardware is introduced.

## 4. Procurement Categories

### 4.1 Base Vehicle Platform

Purpose:

- Provide the physical platform for the battery-only baseline and later hydrogen integration

Needed to start:

- Donor bicycle with frame space suitable for central packaging
- Good brakes
- Structurally sound frame
- Rear dropout compatibility for a hub motor conversion

Selection guidance:

- Prefer a frame with usable triangle or downtube packaging space
- Prefer a platform that can tolerate additional prototype weight
- Avoid highly integrated proprietary e-bike frames for the first build

Status:

- Early purchase
- High priority

### 4.2 Electric Baseline Drive System

Purpose:

- Create the stable propulsion baseline before hydrogen is introduced

Needed to start:

- 48V rear hub motor kit
- Compatible motor controller
- Throttle and/or pedal-assist components if used
- Basic display or control interface if needed by the chosen kit

Selection guidance:

- Stay near the locked 500W nominal / 750W peak class
- Favor common components with known replacement paths
- Avoid unusual high-power kits that push the project away from bicycle behavior

Status:

- Immediate purchase
- High priority

### 4.3 Buffer Battery

Purpose:

- Power the baseline e-bike and later serve as the hybrid buffer battery

Needed to start:

- 48V nominal lithium-ion battery pack
- Compatible charger
- Mounting approach suitable for temporary baseline installation, then later integration refinement

Selection guidance:

- Choose a battery with clear specifications and protection behavior
- Favor reliability and documentation over maximum energy density
- Keep capacity moderate; the first need is stable integration, not long standalone range

Status:

- Immediate purchase
- High priority

### 4.4 Bench Electrical Integration Tools

Purpose:

- Support Phase 2 hybrid power validation before frame integration

Needed to start:

- Stable work surface
- DC wiring materials and connectors appropriate to the 48V system
- Fuse or protection elements for bench testing
- Switch or disconnect means for controlled bench power work
- Representative load approach for bench validation

Selection guidance:

- Bench setup should be organized, repeatable, and easy to inspect
- Avoid improvised wiring that will make results ambiguous

Status:

- Immediate purchase
- High priority

### 4.5 Measurement and Diagnostic Tools

Purpose:

- Make tests meaningful and repeatable

Needed to start:

- Multimeter
- Clamp meter if available
- Temperature observation method
- Basic labeling and note-taking materials

Strongly recommended:

- A simple way to log repeated test observations consistently
- Photo or video capture of test setups and outcomes

Status:

- Immediate purchase
- High priority

### 4.6 General Workshop and Fabrication Basics

Purpose:

- Support repeatable assembly, mockups, and safe rework

Needed to start:

- Standard bike tools
- Standard hand tools for electrical and mechanical work
- Wire management materials
- Mockup materials such as cardboard, foam board, or similar
- Basic bracket and mounting prototype materials

Selection guidance:

- The goal is not full fabrication capability on day one
- The goal is enough tooling to mock, mount, inspect, and revise without guesswork

Status:

- Immediate purchase if missing
- High priority

### 4.7 Safety Equipment

Purpose:

- Support safe bench work and early commissioning discipline

Needed to start:

- Eye protection
- Hand protection appropriate to the work being done
- Fire extinguisher appropriate for workshop electrical risk
- Clear emergency shutdown plan for powered tests

Needed before hydrogen commissioning:

- Hydrogen-appropriate handling procedures
- Ventilation-aware test setup
- Clear inspection and shutdown checklist

Selection guidance:

- Do not defer safety basics because the project is still "only in prototype"

Status:

- Immediate purchase for general safety items
- Hydrogen-specific additions later, before commissioning

### 4.8 Hydrogen Subsystem Hardware

Purpose:

- Enable hydrogen storage, regulation, and fuel-cell integration

Needed later, not first:

- Commercially rated removable hydrogen cartridge solution
- Compatible pressure regulation hardware
- PEM fuel cell in the locked 400W class
- Hydrogen path fittings and interfaces that are rated and documented
- Cartridge dock mechanical components

Selection guidance:

- Buy this only when the battery baseline and bench workflow are ready
- Do not buy exotic or custom pressure-bearing parts early
- Favor traceable, commercially documented components over cheap unknown hardware

Status:

- Delayed purchase
- Critical, but not first

### 4.9 Packaging and Enclosure Materials

Purpose:

- Move from bench integration toward a bicycle-like integrated prototype

Needed later:

- Mounting hardware
- Bracket materials
- Enclosure mockup materials
- Cleaner prototype-grade routing and protection parts

Selection guidance:

- Use mockups before finalizing dimensions
- Keep service access and shutoff access visible in every packaging choice

Status:

- Delayed purchase
- Medium priority until dimensions are better understood

## 5. Buy Now vs Buy Later

### Buy Now

- Donor bicycle
- Rear hub motor kit
- 48V battery and charger
- Bench wiring and protection basics
- Measurement tools
- Workshop hand tools and bike tools
- Mockup materials
- General safety equipment

### Buy After Baseline Bike Works

- Better bench fixtures for hybrid integration
- Packaging-specific mounting hardware
- Cleaner electrical routing materials

### Buy After Bench Workflow Is Stable

- Fuel cell
- Hydrogen cartridge solution
- Regulator and hydrogen path hardware
- Cartridge dock hardware

### Do Not Buy Early

- Custom tanks
- Custom fuel-cell hardware
- High-polish enclosures
- Investor-demo cosmetics
- Production-style industrial design parts

## 6. What Will Actually Create Early Progress

The first real signs of progress are not hydrogen-specific yet. They are:

- A stable battery-only e-bike baseline
- A clean bench-test area with repeatable wiring and measurements
- A running test log workflow
- A packaging mockup that shows where the cartridge, fuel cell, and battery could go

Those four things create leverage for everything after them.

## 7. Suggested First Purchase Order

### Order 1 - Foundation

- Donor bike
- Bike tools and workshop basics
- Safety basics
- Mockup materials

### Order 2 - Electric Baseline

- Rear hub motor kit
- 48V battery
- Charger
- Electrical connectors and protection basics

### Order 3 - Bench Validation

- Measurement tools if still missing
- Better bench wiring and disconnect hardware

### Order 4 - Hydrogen Integration

- Rated cartridge and regulation hardware
- Dock-related hardware

### Order 5 - Packaging Refinement

- Brackets
- Enclosure materials
- Routing cleanup materials

## 8. Minimum Starter Set

If budget or momentum forces a narrow start, the minimum practical starter set is:

- One donor bike
- One 48V rear hub kit
- One 48V battery and charger
- One multimeter
- Basic hand and bike tools
- Mockup materials
- Basic bench wiring and protection hardware
- Safety glasses and workshop fire protection

That is enough to begin Phase 1 and start logging real progress.

## 9. First 30 Days of Useful Progress

If procurement is done in the right order, the first month should aim to produce:

- A documented donor-bike decision
- A purchased and installed baseline drive system
- A first powered battery-only ride test
- A test log started with real entries
- An initial frame packaging mockup

That would be meaningful project progress, not just planning.

## 10. Common Early Procurement Mistakes

Avoid these:

- Buying hydrogen hardware before the bench workflow exists
- Buying high-power drivetrain parts that distort the product target
- Buying custom packaging materials before dimensions are mocked
- Buying "cool" parts with weak documentation
- Letting convenience override safety-critical rating requirements
