# Project Tidus

Project Tidus is an R&D effort focused on a consumer-viable hydrogen-powered bicycle system.

The current repository is centered on early definition and planning documents that lock the product scope, prototype architecture, system interfaces, and build strategy.

## Working Principles

- Bottleneck-first execution
- Test-driven development mindset
- Stability over speed
- Consumer viability over technical novelty

## Current Phase

The project is currently in concept and prototype planning. The next likely artifacts are:

- A subsystem procurement sheet
- A phase-by-phase build checklist
- A safety and test checklist
- A reusable test log workflow
